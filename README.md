# CH-2-Commerce

Java 콘솔 기반 커머스 플랫폼 프로젝트입니다.  
사용자는 카테고리를 선택해 상품을 조회하고, 장바구니에 상품을 담은 뒤 주문할 수 있습니다.  
또한 관리자 모드를 통해 상품을 추가, 수정, 삭제하는 기능까지 확장하는 것을 목표로 합니다.

---

## 1. 프로젝트 소개

이 프로젝트는 객체지향 설계를 연습하기 위해 만든 **콘솔 기반 쇼핑몰 프로그램**입니다.

사용자는 다음과 같은 흐름으로 프로그램을 사용할 수 있습니다.

- 카테고리 선택
- 상품 조회
- 장바구니 담기
- 장바구니 확인
- 주문 확정
- 주문 취소

현재는 **LV1 장바구니 및 주문 기능**까지 구현된 상태입니다.

---

## 2. 개발 환경

- Java 17
- IntelliJ IDEA
- Git / GitHub

---

## 3. 실행 방법

1. 프로젝트를 clone 합니다.
2. IntelliJ IDEA 또는 Java 실행 환경에서 프로젝트를 엽니다.
3. `Main.java`를 실행합니다.

---

## 4. 주요 기능

### 사용자 기능
- 카테고리별 상품 조회
- 상품 상세 정보 확인
- 장바구니 추가 확인 / 취소
- 장바구니 내역 확인
- 총 주문 금액 계산
- 주문 확정
- 주문 취소

### 재고 관리
- 장바구니에 담을 때 재고 확인
- 주문 완료 시 재고 차감
- 재고 부족 시 경고 메시지 출력

---

## 5. 프로그램 흐름

### 메인 메뉴
- 카테고리 선택
- 종료
- 장바구니에 상품이 있는 경우 주문 관리 메뉴 출력

### 상품 선택
- 상품 정보 출력
- 장바구니 추가 여부 확인
    - 확인
    - 취소

### 주문 관리
- 장바구니 확인
- 총 주문 금액 확인
- 주문 확정
- 주문 취소

---

## 6. 객체 구조

### Main
프로그램 실행을 담당합니다.

### CommerceSystem
전체 프로그램 흐름을 제어합니다.
- 메인 메뉴 출력
- 카테고리 / 상품 선택 흐름 처리
- 장바구니 / 주문 흐름 처리

### Category
카테고리 이름과 해당 카테고리의 상품 목록을 관리합니다.

### Product
상품의 이름, 가격, 설명, 재고를 관리합니다.

### Customer
고객 정보와 장바구니를 관리합니다.

### Cart
장바구니 전체를 관리합니다.
- 상품 추가
- 총 금액 계산
- 장바구니 비우기

### CartItem
장바구니에 담긴 개별 상품 정보를 관리합니다.
- 상품
- 수량
- 합계 금액 계산

---

## 7. 실행 예시

- 사용자는 전자제품 카테고리를 선택할 수 있습니다.
- iPhone 15를 장바구니에 담을 수 있습니다.
- 장바구니에서 주문 확정을 할 수 있습니다.
- 주문 완료 시 재고가 감소합니다.
- 주문 취소 시 장바구니가 초기화됩니다.
